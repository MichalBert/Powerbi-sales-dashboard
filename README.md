# Power BI – Dashboard pro analýzu prodejů

Tento projekt představuje interaktivní Power BI dashboard vytvořený pro analýzu prodejů podle produktů, zemí a časových období. Cílem bylo procvičit práci s Power BI, vizualizacemi, datovým modelem a základními DAX measure, a zároveň vytvořit ukázku práce vhodnou do portfolia.

## Hlavní funkce dashboardu
- KPI karty: Total Sales, Total Profit, Units Sold, Average Discount  
- Interaktivní filtry (slicery) pro výběr roku a země  
- Vizualizace:  
  - sloupcový graf pro prodeje podle produktů  
  - line chart pro vývoj prodejů v průběhu roku  
  - mapa zobrazující prodeje podle zemí  
  - graf zisku podle zemí  
- Přehledné rozložení vizuálů a jednotný styl

## Použitá data
Dataset obsahuje informace o:
- prodejích jednotlivých produktů  
- zemích (USA, Kanada, Mexiko, Francie, Německo)  
- zisku, slevách a počtu prodaných jednotek  
- časových údajích (měsíce, roky)
